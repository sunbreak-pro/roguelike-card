# roguelike-card
ローグライク系のターン制カードバトル

## 🎮 ゲームについて

このゲームは、ターン制のカードバトルゲームです。  
プレイヤーは冒険者となり、塔の各階層に現れる敵を倒しながら、最上階のボスを目指します。

### ゲームの特徴

- **ターン制バトル**: 毎ターン、手札からカードを選んで戦います
- **デッキ構築**: 攻撃、防御、スキルカードを組み合わせて戦略を立てます
- **ローグライク要素**: 敵を倒してゴールドを獲得し、強化していきます

## 🚀 遊び方

### 必要環境

- Python 3.7 以上

### ゲームの起動

```bash
python main.py
```

### 基本操作

1. **カードを使う**: カード番号を入力 (1, 2, 3, ...)
2. **ターン終了**: `e` または `end` を入力
3. **ゲーム終了**: `q` または `quit` を入力

### ゲームのルール

1. 毎ターン開始時にエネルギーが3回復し、カードを5枚引きます
2. エネルギーを消費してカードを使用します
3. 「ターン終了」を選ぶと敵のターンになります
4. 敵のHPを0にすれば勝利！
5. 自分のHPが0になるとゲームオーバー

### カードの種類

| 種類 | 説明 |
|------|------|
| 🗡️ 攻撃カード | 敵にダメージを与える |
| 🛡️ 防御カード | ブロック（シールド）を獲得 |
| ✨ スキルカード | 回復や特殊効果 |

### ブロックについて

- ダメージを受けるとき、先にブロックで軽減されます
- ブロックは毎ターン開始時にリセットされます

### 敵のインテント

- 敵の次の行動が表示されます
- 攻撃してくる場合は防御を検討しましょう

## 📖 コードの解説

### 初学者向けの特徴

このコードは、初学者の方にも理解しやすいよう、以下の点に配慮しています：

1. **詳細なコメント**: 各ファイル、クラス、メソッドに丁寧な説明を記載
2. **日本語での解説**: 変数名や関数名の意味を日本語で説明
3. **段階的な学習**: 基本的な概念から応用まで順を追って学べる構成

### 学べる概念

- **クラス（class）**: オブジェクト指向プログラミングの基本
- **継承（inheritance）**: コードの再利用と拡張
- **モジュール分割**: 大きなプログラムを整理する方法
- **ゲームループ**: ゲーム開発の基本パターン
- **状態管理**: ステートマシンの考え方

### 各ファイルの詳細

#### `game/card.py`
カードの基本クラスと各種カード（攻撃、防御、スキル）を定義しています。
継承の使い方を学べます。

#### `game/character.py`
プレイヤーと敵キャラクターを定義しています。
HP、ブロック、エネルギーなどのステータス管理を学べます。

#### `game/battle.py`
ターン制バトルの仕組みを実装しています。
ゲームループとユーザー入力の処理を学べます。

#### `game/game_manager.py`
ゲーム全体の流れを管理しています。
状態遷移（ステートマシン）の考え方を学べます。
