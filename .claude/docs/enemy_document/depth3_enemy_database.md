# 深度3（混沌 - Chaos）敵データベース

## 設計コンセプト

```
【難易度方針】
- 深度1の1.5倍スケーリング
- 連携行動の本格化（Role分担）
- 自己バフの多用
- 法則破綻・矛盾の表現
- 9〜13回のバトルを想定
- フロアボスは変則的な行動パターン

【世界観：混沌】
「腐食を身に纏えば、狂乱に身を置き、混沌を知り」
秩序の崩壊、物理法則の破綻
因果が逆転し、矛盾が共存する不可能な領域
```

## まとめ

### 深度3の設計意図
```
【新規要素】
- 連携行動（Role分担）
- 自己バフの重要性
- 貫通攻撃の脅威
- 形態変化による戦術変化

【学習要素】
- 敵の行動パターン把握
- 連携の崩し方（優先順位）
- Guardの重要性
- 速攻vs持久戦判断
- デバフ管理

【難易度】
深度1の1.5倍
予測可能なパターン
連携で戦術的深み
運要素は排除
```

**プレイヤー体験**
1回目: 連携行動に翻弄され、貫通攻撃で全滅
2回目: 優先順位を理解、形態を見極めて対応
3回目: パターンを把握し、効率的にボス撃破

「混沌を知り」- 秩序の崩壊、しかし予測可能な法則は残る
