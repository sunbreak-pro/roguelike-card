# 深度4（虚無 - Void）敵データベース

## 設計コンセプト

```
【難易度方針】
- 深度1の2.0倍スケーリング
- アーマー貫通攻撃の本格化
- 高火力魔法（魔力×8.0）
- 存在の消失・侵食の表現
- 10〜14回のバトルを想定
- フロアボスは存在消去の脅威

【世界観：虚無】
「虚無感に苛まれた者は深淵を覗く」
存在そのものが消えゆく領域
全てが薄れ、消え、無に還る
色も音も光も失われた世界
```

## まとめ

### 深度4（虚無）の設計意図
```
【新規要素】
- 存在侵食（最大HP減少）
- アーマー貫通の本格化（30〜70%）
- 高火力魔法（魔力×8.0）
- デッキ侵食（枚数減少）
- 即死級の攻撃

【学習要素】
- 貫通対策（AP回復重要）
- 速攻の重要性（侵食対策）
- デッキ枚数管理
- HP閾値管理（即死回避）
- 魔法vs物理の選択

【難易度】
深度1の2.0倍
貫通で防御無視
侵食で長期戦不可
即死で緊張感
```

**プレイヤー体験**
1回目: 貫通攻撃でAP崩壊、侵食で弱体化、全滅
2回目: 速攻戦略採用、AP回復重視、ボス第2フェーズで敗北
3回目: デッキ30枚以上、魔法主体、HP30%死守、ボス撃破

「虚無感に苛まれた者は深淵を覗く」
- 存在が消えゆく恐怖
- 全てが失われる絶望
- 最後の試練への入口
