## 【必須】死亡時成長要素の再設計

### **1. 探索記録 → 次元の図書館**

#### **基本システム**
```
【記録内容】
倒した敵: 完全情報(HP・行動パターン・ドロップ)
出会った敵: 制限情報(外見・基本HP・ドロップのみ)
未遭遇の敵: 情報なし
```

#### **図書館の機能**
```
敵図鑑:
- 全敵の記録閲覧
- 倒した敵は行動パターン・弱点表示
- 出会っただけの敵は基本情報のみ

検索機能:
- 深度別
- カテゴリ別(通常 / エリート / ボス)
- ドロップアイテム別

戦略ノート:
- 自分で書き込み可能
- 攻略メモを残せる
```

#### **ゲームプレイへの影響**
```
図書館で情報を確認してから挑戦可能
→ 敵の行動パターンを予習
→ 対策カードを選択
→ 適切なビルド構築
```

### **3. 装備の痕跡 → アクセサリスロット開放**

#### **基本システム**
```
装備スロット6つのうち、アクセサリは初期ロック
装備の痕跡を集めて段階的に解放
```

#### **解放段階**
```
痕跡10個: アクセサリスロット1 開放
痕跡30個: アクセサリスロット2 開放(合計2つ)
痕跡60個: アクセサリスロット3 開放(合計3つ)
痕跡100個: 全アクセサリスロット同時装備可能
```

#### **アクセサリの特性**
```
アクセサリは他装備と異なり:
- HP/ATK/DEFボーナスなし
- 特殊能力のみ
- 複数装備で相乗効果
- 才能カードとのシナジー
```

#### **アクセサリ例**
```
魔力の指輪: エナジー+1
深淵のコンパス: 隠し部屋発見率+30%
不屈の意志: HP50%以下で自動シールド10(1回/戦闘)
才能増幅器: 才能カード効果+30%
カード選定者: 初期手札+1 / ドロー時選択肢+1
魔石収集者: 魔石ドロップ率+20%
```
