# 新キャラクターシステム完全設計

## 全体コンセプト

### 自由度の高いビルドシステム

- カードの組み合わせでプレイスタイルが変化
- 高速型、カウンター型、重厚型など多様なビルドが可能
- 各キャラクターに固有の仕様・アビリティで差別化

### 熟練度と進化システム

```
カード使用回数に応じて熟練度上昇
1Lv → 5Lvまで育成
5Lvで才能カード解放（新カード獲得）

【必要使用回数】
コスト0: 5→10→15→20 (合計50回)
コスト1: 4→8→12→16 (合計40回)
コスト2: 3→6→9→12 (合計30回)
コスト3+: 2→4→6→8 (合計20回)
```

### 称号システム

```
カード獲得枚数に応じて称号が変化

【剣士系】
0枚: 見習い剣士
5枚: 剣士
15枚: 剣豪
30枚: 剣聖
50枚: 剣神

【魔術士系】
0枚: 見習い魔術士
5枚: 魔術士
15枚: 魔導師
30枚: 大魔導師
50枚: 魔神

【召喚士系】
0枚: 見習い召喚士
5枚: 召喚士
15枚: 上級召喚士
30枚: 召喚師
50枚: 召喚神
```

## キャラクター 1: 剣士（Blade Master）

### 基本コンセプト

**「連撃と勢いで圧倒する前衛戦士」**

- シンプルビルド
- 剣気消費で大ダメージ

### 基礎ステータス

```
HP: 100
攻撃力(ATK): +30%
防御力(DEF): +10%
魔法補正: +0%
初期エナジー: 3
```

### 固有アビリティ: 【剣気(Sword Energy)】

#### 基本仕様

```
【剣気ゲージ】
最大値: 10
初期値: 0

【剣気の蓄積】
物理攻撃カード使用時、剣気が蓄積
- 0コスト: +1剣気
- 1コスト: +1剣気
- 2コスト: +2 or +3剣気
- 3コスト以上: +3剣気
- 剣気蓄積専用カード(1コスト): +4剣気

特殊効果:
- 剣気5以上: クリティカル率+20%
- 剣気8以上: 物理攻撃に貫通+30%
- 剣気10(最大): 次の物理攻撃が確定クリティカル+貫通50%

消費効果:
【基本消費】
- 剣気3消費: 中火力技
- 剣気5消費: 高火力技

【全消費】(例)
- 剣気全消費(10): 超高火力 + 自身のbuff、相手へのdebuff
```

## キャラクター 2: 魔術士（Spell Caster）

### 基本コンセプト

**「多彩な魔法で敵を翻弄する後衛魔導師」**

- 属性の組み合わせでシナジー
- 状態異常を駆使して戦う
- 戦略的で計算されたプレイスタイル

### 基礎ステータス

```
HP: 80
攻撃力(ATK): +0%
防御力(DEF): +5%
魔法補正: +40%
初期エナジー: 4
```

### 固有アビリティ: 【魔力共鳴(Mana Resonance)】

#### 基本仕様

```
【魔力共鳴システム】
同じ属性の魔法を連続使用すると共鳴が蓄積
共鳴レベル: 0 → 1 → 2(最大) ※レベル２以降の共鳴はレベル2の効果が適用される.
異なる属性を使用すると共鳴リセット

【共鳴の蓄積】
- 同属性1回目: 共鳴レベル0
- 同属性2回目: 共鳴レベル1
- 同属性3回目以降: 共鳴レベル2(最大)
- 同属性4回目以降: 共鳴レベル2の効果が発動

【共鳴レベルの効果】
各属性ごとに独立した状態異常と強化ボーナス

火属性:
- レベル0: 通常効果のみ
- レベル1: 火傷スタック+1 / 火魔法威力+15%
- レベル2: 火傷スタック+2 / 火魔法威力+30%

氷属性:
- レベル0: 通常効果のみ
- レベル1: speedDown+1ターン付与 / 氷魔法威力+15%
- レベル2: speedDown+2ターン付与 / 氷魔法威力+30%

雷属性:
- レベル0: 通常効果のみ
- レベル1: よろめき+1ターン付与 / 雷魔法威力+15%
- レベル2: よろめき+2ターン付与 / 雷魔法威力+30%

闇属性:
- レベル0: 通常効果のみ
- レベル1: 脱力+1ターン付与 / 与ダメージの15%HP吸収
- レベル2: 脱力+2ターン付与 / 与ダメージの30%HP吸収

光属性:
- レベル0: 通常効果のみ
- レベル1: デバフ1解除 / ガード+10%
- レベル2: デバフ2解除 / ガード+20%
```

## キャラクター 3: 召喚士（Summoner）

### 基本コンセプト

**「召喚獣と共に戦う戦術家」**

- 召喚獣を場に出して戦う
- 召喚獣ごとに異なる効果
- 複雑で戦略性の高いプレイスタイル

### 基礎ステータス

```
HP: 90
攻撃力(ATK): +10%
防御力(DEF): +15%
魔法補正: +20%
初期エナジー: 4
```

### 固有アビリティ: 【召喚システム(Summoning System)】

#### 基本仕様

```
【召喚獣の基本動作】
召喚された次のターンから召喚獣は行動できる
召喚獣は基本的に一つの行動しかできない
召喚獣の特性は召喚されたターンから発動する
(特性：盾型であれば召喚された瞬間にguardを対象の召喚士と召喚獣に貼る)


【召喚スロット】
基本: 最大2体まで同時召喚可能
拡張可能: 特定カードで3体まで拡張(1戦闘のみ)

【召喚の方法】
召喚カード使用 → 召喚獣が場に出る
召喚獣はターン終了時に自動で行動

【犠牲システム(Sacrifice)】
召喚獣を犠牲にして強力な効果を発動
- 犠牲カード使用 → 指定した召喚獣を消滅させる
- 召喚獣の残りHPに応じて効果増加
- 犠牲後は召喚枠が空く

犠牲効果例:
- 犠牲爆発: 召喚獣HP×2のダメージ
- 犠牲回復: 召喚獣HP分を召喚士が回復
- 犠牲強化: 召喚獣HPに応じて他召喚獣強化

【召喚の解除】
- 召喚獣のHP0
- 犠牲として消費
- 勝利 or 敗北

【召喚獣の種類と特性】
合計12種類
- 物理攻撃型×3
- 魔法攻撃型×3
- 防御型×2
- 支援型×2
- 特殊型×2
```

固有アビリティが確定したら、次は各キャラクターの専用カード設計に移ります:

   - 物理攻撃型
   - 防御型

2. **魔術士の専用カード 30 枚**

   - 各属性魔法 ×6
   - 共鳴強化型
   - 属性コンボ型

   - 召喚カード ×12
   - 召喚強化カード
   - 召喚補助カード
