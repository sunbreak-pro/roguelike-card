```

## 各キャラクターでの使い方

### 剣士での共通カード活用
```
【物理攻撃カード】
- 打撃/連続攻撃: 剣気+1蓄積
- 重撃: 剣気ダメージ+出血
- 全体斬り: 剣気+範囲攻撃

【防御カード】
- 防御/バリア: 剣気を維持しながら守る
- カウンター: 被ダメージで剣気蓄積+反撃

【補助カード】
- 気力回復: エナジー確保で高コストカード
- 強化: 剣気ダメージも倍増
```

### 魔術士での共通カード活用
```
【魔法攻撃カード】
- 魔法弾: 無属性なので共鳴リセット注意
- 貫通魔法: シールド対策
- 集中魔法: 高火力

【防御カード】
- 防御/バリア: 共鳴維持しながら守る
- 回避: 魔術士の低HPをカバー

【補助カード】
- 浄化: デバフ解除(光属性以外)
- 集中: 手札確保+火力強化
```

### 召喚士での共通カード活用
```
【魔法攻撃カード】
- 魔法弾: 召喚獣と合わせて攻撃
- 魔法爆発: 範囲攻撃
- 魔力の雨: 継続ダメージ設置

【防御カード】
- 防御/バリア: 召喚士本体を守る
- 鉄壁: 長期戦で召喚獣維持

【補助カード】
- 気力回復: 維持コスト対策
- 回復: 召喚士本体のHP確保
- 強化: 召喚士の攻撃力強化(召喚獣依存攻撃に影響)
```

## 共通カードの取得タイミング

### 序盤(Depth 1-2)
```
推奨カード:
- 打撃: 基本攻撃
- 防御: 基本防御
- 回復: 基本回復
- 気力回復: リソース確保

理由: シンプルで使いやすい
専用カードが少ない序盤の主力
```

### 中盤(Depth 3-4)
```
推奨カード:
- 貫通攻撃/貫通魔法: シールド対策
- 連続攻撃: 手数増加
- バリア: 長期防御
- 浄化: デバフ対策

理由: 敵が強化されてくる
状況に応じた対策カード
```

### 終盤(Depth 5+)
```
推奨カード:
- 重撃/集中魔法: 高火力
- 鉄壁: 最強防御
- 強化: バフ特化
- 全体斬り/魔法爆発: 範囲攻撃

理由: ボス戦対策
高火力と耐久の両立
```

### 全キャラクター共通
```

6. 貫通攻撃 → 完全貫通攻撃(威力28+貫通70%)

【優先度低】
7. 連続攻撃 → 高速連続攻撃(12×5回)
8. 重撃 → 超重撃(威力55+出血3)
9. バリア → 絶対バリア(シールド+50+被ダメージ-40%)
```

```
【剣士の場合】

【魔術士の場合】
専用カード: 25枚(5属性×5枚)
