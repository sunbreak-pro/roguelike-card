# 新キャラクターシステム完全設計

## 全体コンセプト

### 自由度の高いビルドシステム
- カードの組み合わせでプレイスタイルが変化
- 高速型、カウンター型、重厚型など多様なビルドが可能
- 各キャラクターに固有の仕様・アビリティで差別化

### 熟練度と進化システム
```
カード使用回数に応じて熟練度上昇
1Lv → 5Lvまで育成
5Lvで才能カード解放（新カード獲得）

【必要使用回数】
コスト0: 5→10→15→20 (合計50回)
コスト1: 4→8→12→16 (合計40回)
コスト2: 3→6→9→12 (合計30回)
コスト3+: 2→4→6→8 (合計20回)
```

### 称号システム
```
カード獲得枚数に応じて称号が変化

【剣士系】
0枚: 見習い剣士
5枚: 剣士
15枚: 剣豪
30枚: 剣聖
50枚: 剣神

【魔術士系】
0枚: 見習い魔術士
5枚: 魔術士
15枚: 魔導師
30枚: 大魔導師
50枚: 魔神

【召喚士系】
0枚: 見習い召喚士
5枚: 召喚士
15枚: 上級召喚士
30枚: 召喚師
50枚: 召喚神
```

## キャラクター1: 剣士（Blade Master）

### 基本コンセプト
**「連撃と勢いで圧倒する前衛戦士」**
- 物理攻撃に特化
- カードを使うほど強くなる
- 攻撃的で爽快感のあるプレイスタイル

### 基礎ステータス
```
HP: 100
攻撃力(ATK): +30%
防御力(DEF): +10%
魔法補正: +0%
初期エナジー: 3
```

### 固有アビリティ: 【剣気(Sword Energy)】

#### 基本仕様
```
【剣気ゲージ】
最大値: 10
初期値: 0

【剣気の蓄積】
物理攻撃カード使用時、剣気が蓄積
- 0コスト: +1剣気
- 1コスト: +1剣気
- 2コスト: +2 or +3剣気
- 3コスト以上: +3剣気
- 剣気蓄積専用カード(1コスト): +4剣気

【剣気の効果】
直接ダメージ上昇:
- 物理攻撃ダメージ = 基本威力 + (剣気×5)
- 例: 斬撃(威力10) + 剣気6 = 10+(6×5) = 40ダメージ

特殊効果:
- 剣気5以上: クリティカル率+20%
- 剣気8以上: 物理攻撃に貫通+30%
- 剣気10(最大): 次の物理攻撃が確定クリティカル+貫通50%

消費効果:
【基本消費】
- 剣気3消費: 中火力技
- 剣気5消費: 高火力技

【全消費】
- 剣気全消費(10): 超高火力 + 出血3スタック付与
- 出血: 毎ターン終了時スタック×5ダメージ / 3ターン持続
```

#### 戦闘例
```
ターン1: 斬撃(威力10、剣気+1) × 3回
→ 剣気0→1→2→3
→ ダメージ: 10+0, 10+5, 10+10 = 35ダメージ

ターン2: 強撃(威力20、コスト2、剣気+2)
→ 剣気3→5
→ ダメージ: 20+(3×5) = 35ダメージ
→ 剣気5以上でクリティカル率+20%

ターン3: 剣気集中(1コスト、剣気+4、攻撃なし)
→ 剣気5→9

ターン4: 連撃(威力8×2回、剣気+1)
→ 剣気9→10
→ ダメージ: 8+(9×5), 8+(10×5) = 53+58 = 111ダメージ
→ 剣気10で次回確定クリティカル

ターン5: 剣気全解放(剣気10全消費)
→ 基本威力30 + (10×5)×3 = 30+150 = 180ダメージ
→ 出血3スタック付与(毎ターン15ダメージ×3ターン)
→ 剣気0にリセット
```

### 固有アビリティ: 【魔力共鳴(Mana Resonance)】

#### 基本仕様
```
【魔力共鳴システム】
同じ属性の魔法を連続使用すると共鳴が蓄積
共鳴レベル: 0 → 1 → 2(最大)
異なる属性を使用すると共鳴リセット

【共鳴の蓄積】
- 同属性1回目: 共鳴レベル0
- 同属性2回目: 共鳴レベル1
- 同属性3回目以降: 共鳴レベル2(最大)

【共鳴レベルの効果】
各属性ごとに独立した状態異常と強化ボーナス

火属性:
- レベル0: 通常効果のみ
- レベル1: 火傷スタック+1 / 火魔法威力+15%
- レベル2: 火傷スタック+2 / 火魔法威力+30% / 火魔法が範囲化

氷属性:
- レベル0: 通常効果のみ
- レベル1: 凍結2ターン付与 / 氷魔法威力+15%
- レベル2: 凍結3ターン付与 / 氷魔法威力+30% / 凍結中の敵に追加ダメージ+50%

雷属性:
- レベル0: 通常効果のみ
- レベル1: よろめき2ターン付与 / 雷魔法威力+20%
- レベル2: よろめき3ターン付与 / 雷魔法威力+40% / 雷魔法が2体に連鎖

闇属性:
- レベル0: 通常効果のみ
- レベル1: 脱力2ターン付与 / 闇魔法威力+15% / HP吸収+20%
- レベル2: 脱力3ターン付与 / 闇魔法威力+30% / HP吸収+40% / 呪いスタック+2

光属性:
- レベル0: 通常効果のみ
- レベル1: デバフ1解除 / シールド+10 / 光魔法威力+10%
- レベル2: デバフ2解除 / シールド+20 / 光魔法威力+20% / HP回復+10
```

## キャラクター3: 召喚士（Summoner）

### 基本コンセプト
**「召喚獣と共に戦う戦術家」**
- 召喚獣を場に出して戦う
- 召喚獣ごとに異なる効果
- 複雑で戦略性の高いプレイスタイル

### 基礎ステータス
```
HP: 90
攻撃力(ATK): +10%
防御力(DEF): +15%
魔法補正: +20%
初期エナジー: 4
```

### 固有アビリティ: 【召喚システム(Summoning System)】

#### 基本仕様
```
【召喚スロット】
基本: 最大2体まで同時召喚可能
拡張可能: 特定カードで3体まで拡張(1戦闘のみ)

【召喚の方法】
召喚カード使用 → 召喚獣が場に出る
召喚獣はターン終了時に自動で行動
召喚士が死亡しても召喚獣は残る(1戦闘のみ)

【召喚維持コスト】
召喚獣1体につき、ターン開始時エナジー-1
(初期エナジー4で、2体召喚すると毎ターンエナジー2からスタート)

【犠牲システム(Sacrifice)】
召喚獣を犠牲にして強力な効果を発動
- 犠牲カード使用 → 指定した召喚獣を消滅させる
- 召喚獣の残りHPに応じて効果増加
- 犠牲後は召喚枠が空く

犠牲効果例:
- 犠牲爆発: 召喚獣HP×2のダメージ
- 犠牲回復: 召喚獣HP分を召喚士が回復
- 犠牲強化: 召喚獣HPに応じて他召喚獣強化

【召喚の解除】
- 召喚獣のHP0
- 召喚士が手動解除
- 犠牲として消費
- 戦闘終了

【召喚獣の種類と特性】
合計12種類
- 物理攻撃型×3
- 魔法攻撃型×3
- 防御型×2
- 支援型×2
- 特殊型×2
```
