# ダンジョン階層テーマ設定

## 世界観の基本コンセプト

```
深淵へ続くダンジョンは、かつて栄えた古代文明の遺跡。
各階層は探索者の精神崩壊の過程を表現している。

「腐食を身に纏えば、狂乱に身を置き、混沌を知り、
 虚無感に苛まれた者は深淵を覗く」

深度が深まるほど:
- 肉体から精神へ
- 秩序から混沌へ
- 存在から無へと堕ちていく
```

---

## 第 1 層：腐食（Decay）

### テーマコンセプト

```
「生命の終焉と腐敗」

かつて生きていたものが、朽ち果て、腐り、異形と化した階層。
生物的な死の始まり。有機物の崩壊。
```

### 視覚的要素

```
色彩: 灰色、褐色、緑がかった黒
環境: 湿った石壁、腐敗した植物、緑色の瘴気
照明: 薄暗く、腐臭を感じさせる緑がかった光
```

### 敵の特徴

```
種族: 腐敗した動物、病んだ亡者、変異生物
攻撃属性: 毒、出血、酸化
戦術: 単純だが執拗な攻撃、群れを成す

代表的な敵:
- 腐敗の野犬
- 変異した腐食鴉
- 骨の徘徊者
- 影の這いずり
```

### サウンド（参考）

```
BGM: 不穏な弦楽器、低音のドローン
SE: 滴る水音、肉が擦れる音、かすかな呻き声
```

---

## 第 2 層：狂乱（Madness）

### テーマコンセプト

```
「理性の崩壊と狂気の始まり」

腐食に侵された肉体は、次に精神を蝕む。
正気と狂気の境界が曖昧になり、幻覚と現実が混ざり合う。
恐怖、混乱、そして制御不能な感情が支配する階層。
```

### 視覚的要素

```
色彩: 毒々しい紫、不安定な赤、狂気の黄色、暗い緑
環境:
- 歪んだ通路と錯覚を起こす壁
- 脈打つような生物的な床と天井
- 血痕のような赤い染み
- 意味不明な文字が刻まれた壁
- 時々揺れる空間

照明: 明滅する不安定な光、色が変わる照明
特殊効果: 画面の軽い歪み、時折のフラッシュ
```

### 敵の特徴

```
種族: 狂気に堕ちた人間、恐怖の化身、精神を侵す悪夢
攻撃属性: 混乱、恐怖、狂気、精神ダメージ
戦術: 予測困難な行動、精神攻撃、プレイヤーの判断を狂わせる

代表的な敵:
- 狂乱の囚人（理性を失った亡者）
- 恐怖の影（トラウマの具現化）
- 精神崩壊者（自我を失った探索者）
- 狂気の伝道者（狂気を広める存在）
```

### サウンド（参考）

```
BGM: 不協和音、狂ったメロディ、囁き声の重なり
SE: 笑い声、泣き声、叫び声、肉が擦れる音、心音
```

### 環境ギミック

```
- 精神汚染: 戦闘が長引くほど狂気スタック+1
- 幻覚: 10%の確率で偽の敵情報表示
- 不安定な空間: 時々ターン順がランダム変動
```

---

## 第 3 層：混沌（Chaos）

### テーマコンセプト

```
「秩序の崩壊と法則の破綻」

狂気に堕ちた精神は、やがて現実そのものの歪みを知覚する。
物理法則が意味を成さず、因果が逆転し、矛盾が共存する。
存在の基盤そのものが揺らぐ混沌の領域。
```

### 視覚的要素

```
色彩: 全ての色が混ざり合い、分離と融合を繰り返す
環境:
- エッシャー的な矛盾構造
- 重力の方向が不定
- 固体・液体・気体が同時に存在する物質
- 時間の流れが場所ごとに異なる
- 幾何学的に不可能な形状

照明: あらゆる方向から、あらゆる色の光
特殊効果: 画面の回転、反転、分裂、融合
```

### 敵の特徴

```
種族: 混沌の生物、法則違反者、矛盾の化身
攻撃属性: ランダム状態異常、予測不能なダメージ、法則無視
戦術: 完全にランダムな行動、同時に複数の状態、変身

代表的な敵:
- 混沌の使徒（形が定まらない存在）
- 矛盾体（複数の性質を同時に持つ）
- 法則破壊者（物理法則を無視する）
- 変異の塊（戦闘中に進化する）
```

### サウンド（参考）

```
BGM: 全ての音楽ジャンルの断片が混ざり合う
SE: 逆再生、複数の音の同時発生
```

### 環境ギミック

```
- 法則崩壊: カード効果がランダムに変化（10%）
- 時空歪曲: ターン数が増減する
- 因果逆転: ダメージと回復が入れ替わる（稀）
- 重力反転: 画面が上下逆になる
```

---

## 第 4 層：虚無（Void）

### テーマコンセプト

```
「存在の消失と絶対的な空虚」

混沌を経た精神は、ついに「無」を理解する。
意味も、価値も、存在そのものが失われていく。
全てが無に帰し、何も残らない完全な虚無の領域。
```

### 視覚的要素

```
色彩: 深い紫、純粋な黒、虚空の白、存在しない色
環境:
- ほとんど何も存在しない空間
- 浮遊する石の破片（消えかけ）
- 半透明で不安定な床
- 存在が薄れていく建造物の残骸
- 全てが消失の途上にある世界

照明: まばらで弱い光、消えゆく光源
特殊効果: 全てが半透明、徐々にフェードアウト
```

### 敵の特徴

```
種族: 存在希薄な幽霊、虚無の化身、消失寸前の者
攻撃属性: 虚無侵食、存在削減、消失、完全沈黙
戦術:
- 実体がほとんどない（物理攻撃半減）
- 不可視化
- 存在確率の変動

代表的な敵:
- 虚無の彷徨者（ほぼ透明な人型）
- 消失する者（存在率が変動）
- 忘却の化身（全てを忘れさせる）
- 空虚そのもの（虚無の具現化）
```

### サウンド（参考）

```
BGM: ほぼ無音、時折のかすかな音
SE: 消えゆく音、虚無の静寂
```

### 環境ギミック

```
- 存在希薄化: 毎ターン確率でカードが消失
- 虚無侵食: 最大HP-1%（永続、回復不可）
- 完全沈黙: 全てのカード使用不可（稀）
- 記憶消失: 戦闘開始時デッキからランダム削除
```

## 第 5 層：深淵（Abyss）

### テーマコンセプト

```
「原初の闇と終焉の静寂」

すべての終着点。生と死、存在と虚無、全てが収束する場所。
もはや階層ですらなく、概念の終わり。
ここには「死」すら存在しない。ただ「終わり」があるのみ。
```

### 視覚的要素

```
色彩: 純粋な黒、概念としての闇、光の不在
環境:
- 無限に続く暗黒の空間
- 重力の存在しない虚空
- 浮遊する古代の遺物
- 時間の概念が消失した世界
- 「深淵そのもの」が意志を持つ

照明: ほぼ無し。プレイヤー周囲のみかすかな光
特殊効果:
- 完全な暗転
- 画面の端から侵食する闇
- 存在の境界線が消える
```

### 敵の特徴

```
種族: 深淵の化身、終焉の使徒、名前のないもの
攻撃属性: 全ての状態異常、即死級ダメージ、概念攻撃
戦術:
- 学習AI（プレイヤーの行動を記憶）
- 複数回行動
- フェーズ変化
- 全カテゴリの攻撃使用

代表的な敵:
- 深淵の守護者（初回ボス）
- 狂乱の魔神 インサニティ（真の邪神1）
- 虚無の帝王 ナルヴォイド（真の邪神2）
- 終焉の化身 エンドブリンガー（真の邪神3）
```

### サウンド（参考）

```
BGM:
- 序盤: 完全な静寂
- 中盤: かすかな心音のような低音
- 終盤: 全てを飲み込む轟音

SE: 深淵の呼吸音、世界の終わりを告げる鐘
```

### 環境ギミック

```
- 深淵の侵食: 毎ターン最大HP-2%（回復不可）
- 時間停滞: 一部のカード効果が持続しない
- 概念崩壊: デバフが永続化する可能性
- 終焉の予兆: HP20%以下で全能力-50%
```

### 特別ルール

```
❌ 生還手段が全て無効
❌ 転移石使用不可
❌ 途中撤退不可
⚠️ ボスを倒すか死ぬかの二択
```

## 階層間の繋がりと物語性

### 深度の意味

```
深度1（腐食）: 肉体の崩壊
深度2（狂乱）: 精神の崩壊
深度3（混沌）: 秩序の崩壊
深度4（虚無）: 存在の消失
深度5（深淵）: 全ての終焉

→ 「腐食を身に纏えば、狂乱に身を置き、混沌を知り、
   虚無感に苛まれた者は深淵を覗く」
```

### プレイヤーの精神状態の変化

```
深度1: 「体が…腐っていく…」
深度2: 「何が現実だ？　笑いが止まらない！」
深度3: 「上も下も、前も後ろも、全てが同時に存在する」
深度4: 「私は…誰だった？　何も…ない…」
深度5: 「                              」（無音）
```

## 各階層の象徴的なオブジェクト

### 深度 1：腐食

```
- 積み上げられた骨
- 腐敗した遺体
- 緑色に光る苔
- 朽ちた武器防具
```

### 深度 2：狂乱

```
- 血の手形がついた壁
- 意味不明な文字の羅列
- 狂った笑顔の彫刻
- 脈打つ肉の塊
```

### 深度 3：混沌

```
- エッシャーの階段
- 同時に固体と液体の物質
- 矛盾する方向標識
- 存在しない色の光
```

### 深度 4：虚無

```
- 半透明の壁
- 消えかけの文字
- 存在しない扉
- 空虚な空間そのもの
```

### 深度 5：深淵

```
- 古代の祭壇
- 亀裂（邪神への入口）
- 浮遊する遺物
- 闇そのもの
```
