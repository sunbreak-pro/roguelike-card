# 深度2（狂乱 - Madness）敵データベース

## 設計コンセプト

```
【難易度方針】
- 深度1の1.2倍スケーリング
- 重度状態異常の導入（重度混乱、重度恐怖、重度狂気）
- 精神攻撃の本格化
- 予測困難な行動パターン
- 8〜12回のバトルを想定
- フロアボスは狂気管理が最重要

【世界観：狂乱】
「腐食を身に纏えば、狂乱に身を置き」
肉体の崩壊の次は、精神の崩壊
正気と狂気の境界が消え、恐怖と混乱が支配する
```

## 新規状態異常：狂気システム

### 狂気（Insanity）スタック
```
【軽度狂気】
- カードコスト+1（ランダム1枚）
- 10%の確率で行動誤認

【重度狂気】
- カードコスト+2（ランダム2枚）
- 30%の確率で行動誤認
- 手札シャッフル（ターン開始時）

【完全狂気（スタック5）】
- 1ターン完全行動不能
- 全カード使用不可
- HP-10（自傷）
```

### 混乱（Confusion）の強化
```
【軽度混乱】
- 20%誤射（自分へ）
- カードコスト+1（ランダム）

【重度混乱】
- 40%誤射（自分へ）
- カードコスト+2（ランダム）
- 敵味方識別不能
```

### 恐怖（Terror）
```
【軽度恐怖】
- 与ダメージ-20%
- 回避率-20%

【重度恐怖】
- 与ダメージ-40%
- 回避率-40%
- 10%の確率でターンスキップ
```

## まとめ

### 深度2（狂乱）の設計意図
```
【新規要素】
1. 狂気システム（スタック5で完全狂気）
2. 重度デバフ（恐怖・混乱・狂気）
3. 予測不能な行動パターン
4. 精神攻撃特化

【学習要素】
1. 狂気スタック管理
2. デバフ解除の重要性
3. 完全狂気の脅威
4. 魔法攻撃の有効性（物理軽減対策）

【難易度カーブ】
深度1より1.2倍強化
デバフ管理が複雑化
狂気の累積が致命的
フロアボスは完全狂気との戦い
```

### プレイヤー体験
```
1回目: 狂気管理失敗、完全狂気で全滅
2回目: デバフ解除カード用意、狂気3以下維持を意識
3回目: 魔法カード活用、ボス撃破
```
